## Download Manager 
### Use
```
python3 downloadManager.py
```
### Install
```sh
pip install colored bullet
```
### It is necessary to have aria2 installed
* Arch Linux
```sh
sudo pacman -S aria2  
```
* Fedora
```sh
sudo dnf install aria2
```
* Debian
```sh
sudo apt install aria2
```
